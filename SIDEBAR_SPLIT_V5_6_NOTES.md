# Sidebar Split v5.6 — SMM menejer / Mobilograf

Bu versiyada eski tizim, eski API va mavjud sahifalar saqlandi.

Yangilanish faqat menu/sidebar qismiga qilindi:

- Sidebar boshidan 2 ta ish yo‘nalishiga bo‘lindi:
  - SMM menejer
  - Mobilograf
- SMM menejer menyusida kontent, bonus, vazifa, reklama, finance, hisobot, analytics, xodimlar va sozlamalar ko‘rinadi.
- Mobilograf menyusida boshqaruv, safar rejasi, vazifalar, kontent reja, media kutubxona, kunlik hisobot va profil ko‘rinadi.
- Menyu switcher desktop va mobile menu ichida ham ishlaydi.
- Role/permission tizimi buzilmadi: userda ruxsat bo‘lmagan sahifa baribir ko‘rinmaydi.
- Menu hozirgi dark navy ko‘rinishida qoldi.

Build tekshirildi:

```bash
npm run build
```

Natija: muvaffaqiyatli.
