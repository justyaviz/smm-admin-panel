# aloo SMM Panel v3.0

Aloo do‘konlar tarmog‘i uchun Railway-ready SMM boshqaruv paneli.

## Tayyor modullar
- Login va JWT autentifikatsiya
- Dashboard
- Kontent boshqaruvi
- Oylik kontent kalendari
- Kampaniyalar
- Target reklama
- PostgreSQL avtomatik migratsiya
- Railway frontend/backend Docker deploy
