# Railway hotfix

Frontend builddagi `vite: not found` xatosi tuzatildi.

## Railway Root Directory
- frontend: `/frontend`
- backend: `/backend`

## Frontend Build Command
`npm install --include=dev && npm run build`

## Frontend Start Command
`npm start`

Eski `package-lock.json` fayllari ichki registry manzillarini saqlagani uchun olib tashlandi. Railway yangi lockfile yaratadi.
