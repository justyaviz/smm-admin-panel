# Sidebar Split v5.7 — Mobilograf Bonus Menu

Bu versiyada eski tizim, API va funksiyalar saqlangan. Faqat sidebar workspace split ichida Mobilograf yo‘nalishiga **Bonus tizimi / KPI va mukofot** menyusi qo‘shildi.

## O‘zgarish

- SMM menejer menyusi o‘zgarishsiz qoldi.
- Mobilograf menyusiga `Bonus tizimi` qo‘shildi.
- Route, permission va backend logikaga tegilmadi.
- Agar userda bonus bo‘limiga ruxsat bo‘lmasa, mavjud permission tizimi bo‘yicha baribir yashiriladi.

- Boshqaruv markazi
- Safar rejasi
- Vazifalar
- Bonus tizimi
- Kontent reja
- Media kutubxona
- Kunlik filial hisobotlari
