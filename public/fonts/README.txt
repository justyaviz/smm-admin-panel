Gilroy font files are intentionally not bundled in this ZIP.

To enable the exact Gilroy family in the browser, copy your licensed files into this folder with these names:
- Gilroy-Regular.ttf
- Gilroy-Medium.ttf
- Gilroy-Semibold.ttf (or duplicate/rename your closest licensed weight)
- Gilroy-Bold.ttf

The CSS is already configured to load these files when present. Until then, it uses local Gilroy (if installed) and a close system fallback.
